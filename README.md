## React Tasks App

#### What this app can do?

- Create new tasks
- View tasks
- Update tasks
- Delete tasks

#### Features

- Simple and beautiful
- Responsive
- Using gravatar api to fetch user avatar

#### Tehcnologies

- HTML5
- CSS3
- JavaScript
- NodeJS
- ReactJS
- localStorage
